# Календарный график работ — сайт с входом и регистрацией

Полноценный сайт с авторизацией по паролю и регистрацией по номеру телефона. Данные хранятся в облаке (Firebase Firestore) для каждого пользователя.

## Структура

- **index.html** — страница входа и регистрации (email+пароль или номер телефона + СМС).
- **app.html** — приложение «Календарный график» и «Раздел 3»; доступно только после входа.
- **firebase-config.js** — настройки Firebase (нужно заполнить своими данными).

## Настройка Firebase

1. Зайдите на [Firebase Console](https://console.firebase.google.com/).
2. Создайте проект (или выберите существующий).
3. В разделе **Authentication** включите:
   - **Email/Password** (Вход по email и паролю).
   - **Phone** (Вход по номеру телефона; для СМС потребуется настройка в консоли).
4. В разделе **Firestore Database** создайте базу в режиме **тестовый** (или настройте правила доступа).
5. В разделе **Project settings** (⚙️) → **Your apps** добавьте веб-приложение, скопируйте конфиг (apiKey, authDomain, projectId, storageBucket, messagingSenderId, appId).
6. Откройте **firebase-config.js** и подставьте скопированные значения вместо `YOUR_API_KEY`, `YOUR_PROJECT_ID` и т.д.

## Правила Firestore (безопасность)

В Firestore → **Rules** можно использовать, например:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
```

Так каждый пользователь видит и меняет только свои данные.

## Запуск

- Локально: откройте **index.html** в браузере (лучше через локальный сервер, например `npx serve .` в папке site).
- Для входа по телефону и сохранения в Firestore нужен доступ по **https** (или localhost). При открытии файла по `file://` часть функций может не работать — используйте хостинг или локальный сервер.

## Регистрация и вход

- **Вход** — email и пароль или номер телефона (код приходит в СМС).
- **Регистрация** — email и пароль (не менее 6 символов).
- **По телефону** — вкладка «По телефону»: введите номер, нажмите «Отправить код по СМС», введите код и вы будете авторизованы (при первом входе создаётся новый пользователь).

После входа откроется **app.html** с календарным графиком и разделом 3. Данные автоматически сохраняются в Firestore при изменениях.
