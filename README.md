# Календарный график работ и Раздел 3

Веб-приложение для ведения календарного графика и журнала выполнения работ (стройка, монолит, нерудные материалы).

## Возможности

- **Календарный график** — задачи, ответственные, даты, статусы (План / В работе / Завершено / Задержка)
- **Диаграмма Ганта** — наглядный график по срокам
- **Раздел 3** — таблица «Сведения о выполнении работ» с возможностью добавлять строки
- Добавление и удаление ответственных в списке
- Сохранение данных в браузере (localStorage) и **прямо в HTML-файл**
- Экспорт графика в CSV, печать

## Как открыть

Откройте файл `index.html` в браузере (двойной клик или перетащите в окно браузера).

## Технологии

HTML, CSS, JavaScript. Внешние зависимости не нужны.

## Лицензия

Свободное использование.
